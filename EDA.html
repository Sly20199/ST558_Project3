Introduction

Data Description

The Water Potability dataset contains water quality metrics for 3,276 water samples. Each observation includes nine physicochemical properties of the water along with a binary label indicating whether the water is safe for human potable or not.

Purpose

The goal of this exploratory data analysis is to understand the distributions and relationships among the water quality variables, particularly as they relate to Potability(our response variable). We want to identify which physicochemical properties differ between potable and non-potable water and assess whether any strong patterns or correlations exist. This EDA will inform the development of a classification model to predict water potability based on these features.

Split the data

We will use the tidymodels framework to split the data into training and testing sets, and create cross-validation folds on the training sets.

Handle Missing Values Before Splitting

We impute missing values with the column mean using tidyr::replace_na().

water_clean <- water |>
  replace_na(list(
    ph = mean(water$ph, na.rm = TRUE),
    sulfate = mean(water$sulfate, na.rm = TRUE),
    trihalomethanes = mean(water$trihalomethanes, na.rm = TRUE)
  ))
colSums(is.na(water_clean))

Train/Test Split(75/25, Stratified on Potability)

We use intial_split from tidymodels with thestrate argument to ensure that the proportion of potable vs. non-potable is preserved in both the training and testing sets.

set.seed(123)

water_split <- initial_split(water_clean, prop = 0.75, strata = potability)
water_train <- training(water_split)
water_test <- testing(water_split)

water_train |>
  count(potability) |>
